## Installation

**1) Clone repository**

`git clone git@github.com:sharaendy/challenge_boro_frontend.git`

**2) Parcel first**

`npm install parcel`

**3) Establish dependencies**

`npm ci`

**4) Build**

`npm start`	

**5) Finally! Go to ** http://localhost:1234
